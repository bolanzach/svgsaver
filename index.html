A Note for devs: This lib is using ECMAScript6. To make changes, first run npm install in your local repo's directory. Make all your edits within the 'src' directory. To compile your changes, use 'npm run compile'. This compiles the fugly ECMA6 code to ECMA5. If successful, you can see your changes in the browser.js file (in the root of this project). You can run browser.js as a normal js file.
